# 6.8 for 루프 소개

## while loop
* Indefinite loop
    - 진입 조건은 처음에 검사하나, 이후 내용에 대해선 문법적으로 크게 제한이 없음.
    - 원래 목적

* Counting loop의 조건
    - Counter initialization
    - Counter check
    - Counter change
    - 사용하기 까다로운 점이 있고, 읽기 명확하지 않아 에러가 많이 생기긴 함.

## for loop
* while문의 까다로운 점을 보완
### 기본 형태

            for (int i = 1; i <= 10; i++)
                printf("%d", i);
            

* block을 만들 필요 없음. (i++이 조건으로 들어갔기 때문)
* counting loop의 조건이 모두 충족.

* 순서
    - 변수가 선언이 되고, 초기화는 반복 횟수와 상관없이 최초 1번 실행
    - 블록 내 식 수행
    - 증감식 수행
    - 조건 검토
