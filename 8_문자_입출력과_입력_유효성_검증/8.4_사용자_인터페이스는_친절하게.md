# 8.4 사용자 인터페이스는 친절하게

- 인터페이스(interface): 사용자가 프로그램을 이용하도록 하는 사이 부분
- 방법론

- 사용자의 어떤 행동을 할지 예측하고 꼼꼼하게 만들기.

## 예제 1: 개발자들은 알아듣는 프로그램

        int main()
        {
            int count = 0;

            while (1)
            {
                printf("Current count is %d. Continue? (y/n)");

                if (getchar() == 'n')
                    break;

                while (getchar() != '\n')
                    continue;

                count++;

            }
            return 0;
        }

## 예제 2: 예외 상황을 고려한 코드

            int main()
            {
            int count = 0;

            while (1)
            {
                printf("Current count is %d. Continue? (y/n)");

                int c = getchar();

                if (c == 'n')
                    break;
                else if (c == 'y')
                    count++;
                else
                    printf("Please input y or n\n");

                while (getchar() != '\n')
                    continue;
            }
