# 3.5 정수와 실수

## 정수
- 음의 정수(..., -3, -2, -1), 0, 양의 정수(1, 2, 3, ...)
    - 컴퓨터에서 자료형을 만들 땐, 0과 양의 정수를 다루는 자료형과 세 가지 모두를 다루는 자료형으로 구분된다. (signed, unsigned)
    - 예시: 2, -15, 3697
- 내부적으로 2진수

## 실수
- 2.0, 3.16, 2.74, 0.123 등
- 내부적으로 '부동 소수점' 표현법 사용
    - 0.314E1 = 0.314 * 10¹
    - 3.14E123 = 3.14 * 10¹²³
- 3.14 = 3.14E0 = 3.14e0 = 0.314E1 = 31.4E-1
    - 소수점이 둥둥 떠다닌다

- 내부적으로 2진수
