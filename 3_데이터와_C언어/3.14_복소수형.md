# 3.14 복소수형

## 사용
* `complex.h` 삽입으로 사용 가능 (`#include <complex.h>`)
    - visual studio:
            `_Dcomplex z;`
            `z._Val[0] = 1.0;
            `Z._Val[1] = 1.0;
    - gcc: `double _Complex z = 1 + 2*I`
    
* 복소수는 보통 만드는 사람이 직접 만듦.
    - gcc: 실수는 `creal()`, 허수는 `cimag()` 


* 잘 모르겠음, 나중에 필요할 때 공부할 것.
