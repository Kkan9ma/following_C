# 5.3 더하기, 빼기, 부호 연산자들

## triple assignment
`income = salary = bonus = 100 // income = (salary = (bonus = 100))`

## 부호와 관련하여
* `+`: 부가적인 기능은 x

* 실수 + 정수
    - `1.0f + 2`의 결과는 어떤 자료형인가?
        - 실수 자료형이 된다. (promotion)
