* 32비트
    - 1비트: 부호
    - exponent: 몇 제곱인지에 8비트를 사용
    - 분수 부분만 23비트 부분에 저장
    - exponent: 8개 비트 => 총 256개 숫자 저장 가능
        - signed 형태로 저장하는 방법과
        - unsigned 형태로 저장하는 방법이 있음.
        - 제곱하는 숫자로 음수로 저장 가능.
        - 지수도 256개만큼 그만큼 사용 가능하다.
    - fraction
        - 1/2 자리, 1/4 자리.. .. .. 
        - 2의 -1, 2의 -2 ...
