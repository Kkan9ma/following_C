# 9.9 이진수 변환 예제

- 10진수를 2진수로 변환하기

## mine 반복문

            void print_binary_loop(unsigned long n) {

                int dividend = n;
                int remainer = 0;
                int quotient = 0;

                while (dividend > 0)
                {
                    quotient = dividend / 2;
                    remainer = dividend % 2;

                    printf("%d", remainer);

                    dividend = quotient;
                }
                printf("\n");

            }

## prof 반복문

            void print_binary_loop(unsigned long num)
            {
                while (1)
                {
                    int quotient = num / 2;
                    int remainer = num % 2;

                    printf("%d", remainer);

                    num = quotient;

                    if (num == 0) break;
                }

                printf("\n");
            }

## recursion

            void print_binary(unsigned long n)
            {
                int remainer = n % 2;

                if (n >= 2)
                    print_binary(n / 2);

                printf("%d", remainer);

                return;
            }

- 출력이 recursion 이후에 있음.

  - 이전 강의(9.6)에서 설명되어 있듯, 조건문에 걸려서 실행되지 못했던 `printf()`가, `n`이 조건을 만족하지 않게 되자, 들어갔던 값(`remainer`)의 순서대로 실행됨.

- 반복문과 recursion의 결과가 같지 않다?
  - 사실, 이런 문제를 풀 땐, 이후에 배울 배열을 이용하면 반복문으로도 제대로 된 순서로 출력할 수 있음.
